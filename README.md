# QuantumSU2k.jl

High-precision of SU(2)\_k quantum Racah–Wigner symbols. The paramter q is chosen to be a root of unity. 

## Features
- Exact arithmetic via cyclotomic fields
- Quantum integers and factorials
- Racah formula for q-6j symbols, fsymbols, gsymbols, rsymbols, ..
- Designed for Turaev–Viro applications

## Status
Early development (private repository).
